# 🎬 AI Movie Recommendation System

A machine learning-based movie recommender using NLP and cosine similarity.

## 🚀 Features
- Content-based movie recommendation
- Uses real dataset (1000+ entries)
- Interactive UI with Streamlit
- Explanation of recommendations

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 🧠 How It Works
- Converts movie metadata into vectors using CountVectorizer
- Uses cosine similarity to find similar movies
- Displays top recommendations via UI
